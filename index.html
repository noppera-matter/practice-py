① 카드 1장 추가 — 원하는 코스의 items 안에 복사·붙여넣기:
{ code: "코드한줄", desc: "해설", blank: "빈칸단어", hint: "힌트" }
· blank는 code 안에 실제 있는 단어여야 하며, 생략하면 퀴즈에서 제외됩니다
· desc 안에서 <b>굵게</b>, <code>코드체</code> 사용 가능

② 새 소단원 — { section: "이름", items: [ ... ] } 덩어리 추가

③ 새 코스(큰 카테고리) — 아래 덩어리를 통째로 복사해 추가:
{ emoji: "📕", title: "코스이름", intro: "소개", sections: [ ... ] }

저장 → 새로고침하면 학습 카드·퀴즈 카테고리에 자동 반영됩니다.
